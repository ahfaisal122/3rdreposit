How to add files to github

Let's do it

Command lines to push into github after editing code in a project:

  • git add .
  • git commit -m "third commit"
  • git push

this is how you commit changes

more git commands for branches

git branch comment# shows the branches name

git branch image-add comment#to add new branch: git branch 'branch name'

git checkout image-add comment# switch in another branch: git checkout 'branch name'

git add . comment# add the files

git commit -m "added ozil picture" comment# git commited

git push --set-upstream origin image-add comment# push the new files including image-add branch

------------------------------------------------------

Commands for merge branches

git checkout main comment# go to main branch

git merge image-add comment# merge the branch with main: git merge 'branch name'

git push comment# push the files into main: git push

N.B. Any type of change or edit in the code, make sure it is added by git add command then commited and then pushed
